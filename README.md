> Generator for tiny Drupal8 Themes, enjoy.

## Installation

First, install [Yeoman](http://yeoman.io) and generator-ng2-component using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-d8-theme
```

Then generate your new Drupal8 Theme:

```bash
yo d8-theme
```
